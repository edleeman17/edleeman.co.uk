Having GitHub self hosted runners is awesome. It means I can have ‘local’ staging environments that get deployed but all managed by GitHub. 

Using them to deploy a Ghost CMS custom theme to a local instance of Ghost by using GitHub actions to deploy to the local Ghost API. Best of all, it’s a reverse tunnel, I don’t even need to expose the machine
