# Ingest - Capture anything from anywhere

[Project: Ingest](https://edleeman.co.uk/projects/ingest/)

Another silly project coming your way. Probably only useful to me but I am sharing it anyway.

Do you get messages that you need to follow up at some point? Do you need to remember to reply to someone but always forget? Something important you just need to be aware of until you can get back to it?

That’s holding a lot in your head yano. My neurospicy brain forgets things exist a lot of the time.

> Like when you have a WhatsApp notification from someone you’ve just been speaking to, then you get another from the group chat. And it opens Person A’s message then jumps to the unread group chats message. Marking Person A as read even though you haven’t *read* anything is a mild fucking annoyance

I leave notifications un-dismissed so that I remember I need to action them.

That’s where Ingest comes in.

Send *thing* somewhere so that I remember it exists. Display it in a todo style page.

Supports:
- iOS share sheets
- Discord
- Signal (soon)
- REST API (build your own integration)
- Add from UI

The flow for discord for example is to forward a message to a private server channel and a bot picks it up and adds it to Ingest.

Signal, forward to Note to Self

Raycast??? Paste text I guess.

Any raw text or image gets ingested and stored in a Web UI. Within the UI you can group, reorder and check-off.

MIT so go nuts. Also any contributions are more than welcome!

[Check out Ingest here](https://edleeman.co.uk/projects/ingest/)
